file structure

Item come with 4 version with alot of custom setting, choose your version to start by folder name below

preloader

open js/variable.js find

            var __preloaderFadeOut = 1200; // fadeout
            var __preloaderDelay = 800; // delay

change the preloader time, for preloader image, open css/style.css find

            background: #000; /* preloader background color */

change preloader background color

and find

            #preloader-img {
              position: absolute;
              top: 50%;
              left: 50%;
              background: url("../img/preloader.gif") no-repeat center; /* preloader image */
              width: 60px; /* preloader image width */
              height: 60px; /* preloader image height */
              margin-top: -30px; /* half of preloader image height */
              margin-left: -30px; /* half of preloader image width */
            }

preloader image place at img/preloader.gif, follow above comment to change the preloader image size, for example if your preloader image is 100 width, 260 height, then change css like below

              background: url("../img/preloader.gif") no-repeat center; /* preloader image */
              width: 100px; /* preloader image width */
              height: 260px; /* preloader image height */
              margin-top: -50px; /* half of preloader image height */
              margin-left: -130px; /* half of preloader image width */
            }

animation delay

openindex.html find

            data-delay=

you will find result more than one at each section

you can change the delay time, for example

            data-delay="300"

each section first animate element start from deta-delay="0" , i add 50+ delay per element, to add element delay, element need a class name animate and a data name data-delay="", to remove delay just set delay to 0

do not set delay too high, max delay time about 500 for better fadein timing

    background and overlay

    item come with 4 header version

    static background

    open js/variable.js find

                var __staticBackground = true; // set background image by css
            var __videoHeader = false; // video background toggle
            var __slideshowHeader = false; // image background toggle
    set the config just like above, section static header will be enable

    for example, we are going change #home section background now, open css/color.css find

                #home{
              background: url(../img/bg/home.jpg) no-repeat center center / cover; /* home background */
            }

            #home:before {
              background: rgba(0, 0, 0, 0.55); /* home overlay background */
            }

    we have comment the code, normally dont need change the image path, just replace the image with same name, to change overlay, change the :before rgba value, last number is opacity, to disable overlay, set to 0, set highter number for more dark, the opacity valus from 0 to 1

    single static background

    open js/variable.js find

                var __staticBackground = false; // set background image by css
            var __videoHeader = false; // video background toggle
            var __slideshowHeader = true; // image background toggle

    and find

                var __imageNumber = 1; // how many slideshow image to show, 1 for static image

    set to 1 mean single image, image place at img/bg-01.jpg

    slideshow background

    open js/variable.js find

                var __staticBackground = false; // set background image by css
            var __videoHeader = false; // video background toggle
            var __slideshowHeader = true; // image background toggle

    set the config just like above

    and find

                var __imageNumber = 3; // how many slideshow image to show, 1 for static image

    __imageNumber mean how many slideshow image to show, set to more than 1 mean slideshow enable, image place at img/bg-01.jpg if you need more than one image, place image name as bg-02.jpgbg-03.jpgbg-04.jpg...bg-10.jpgbg-11.jpg something like that :)

    and find

                var __slideshowStatic = false; // true for fade effect slideshow, false for css3 effect slideshow
            var __slideshowShuffle = false; // random image
            var __slideshowDelay = 8000; // slideshow delay
            var __slideshowTransitionDuration = 2500; // transition duration
            var __slideshowAnimationDuration = 12000; // animation duration, __slideshowDelay + 2000 for better fadeout timing

    follow the comment to config slideshow effect, and if you want change the effect, find

                var __slideshowAnimationSet = [ // animation effect set
                    'kenburnsUp',
                    'kenburnsDown',
                    'kenburnsLeft',
                    'kenburnsRight'
                  ];
            var __slideshowTransitionSet = [ // transition effect set
                    'fade'
                  ];

    you can change the effect name, for more info, please view vegas.js, the page have live demo to show effect and effect name, contact us if you need any help, thanks :)

    youtube video background

    open js/variable.js find

                var __staticBackground = false; // set background image by css
            var __videoHeader = true; // video background toggle
            var __slideshowHeader = false; // image background toggle

    set the config just like above, video header will be enable

    and find

                var __youtubeUrl = 'https://www.youtube.com/watch?v=ZBVZAeGHIv8'; // youtube video url
            var __videoStartTime = 1; // video start time, 70 mean 70 seconds
            var __videoEndTime = 0; // video end time, 180 mean 180 seconds, 0 mean play to end
            var __videoMute = false; // mute video on start

    change the url to your youtube video link, start time and end time 1 mean 1 seconds, video mute mean mute at video start

    video wont active at mobile, place a image at img/bg/video-mobile.jpg for mobile, and set a image before video loaded, place at img/bg/video-desktop.jpg

    background overlay

    to set overlay (not static image version), please open js/variable.js find

                var __overlay = true; // overlay

    set true to enable overlay

    open css/color.css find

                .overlay {
              background: rgba(0, 0, 0, 0.55); /* overlay background color and opacity for non-css background */
            }

    to change overlay, change the rgba value, last number is opacity, to disable overlay, set to 0, set highter number for more dark, the opacity valus from 0 to 1

    audio

    open js/variable.js find

                var __audio = false; // audio background toggle

    set true to enable background music, audio file place at audio/01.mp3, audio file not include at item

    audio only active when youtube video background disable, because html5 reason, mobile cant play audio auto start, will play with a button.

    subscribe (mailchimp)

    • 1. first you need a mailchimp account, api, and list id, for more info please view http://mailchimp.com/
    • 2. and then find your mailchimp form id, login mailchimp, select your list, click Signforms > Embedded forms > find Copy/paste onto your site
    • 3. now search
                form action=

    you will see something like that

                xxx.xxx.list-manage.com/subscribe/post?u=xxxxxx&id=xxxxxx

    that link is your mailchimp post url, then next step

    open js/variable.js find

                var __mailChimpVersion = false; // mailchimp toggle, if disable, php version will enable
            var __mailChimpUrl = 'MAILCHIMP_POST_URL_HERE'; // mailchimp post url

    set to true to enable mailchimp subscribe form, then edit the url to your mailchimp post url
